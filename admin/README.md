# TaxQue Admin Panel

A comprehensive admin panel for managing TaxQue tax services, built with Next.js and Tailwind CSS.

## Features

- **Service Management**: Create, edit, and manage tax services
- **Category Management**: Organize services by categories
- **Blog Management**: Manage blog posts and content
- **Team Management**: Manage team members and their information
- **Job Management**: Create and manage job postings
- **User Management**: Handle user registrations and applications
- **FAQ Management**: Manage frequently asked questions
- **Price Management**: Set and manage service pricing
- **Order Management**: Track and manage orders
- **Contact Management**: Handle contact forms and leads

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the TaxQue Admin Panel.

## Technology Stack

- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Redux Toolkit
- **UI Components**: Custom components with Tailwind
- **Rich Text Editor**: Quill.js
- **Notifications**: React Toastify
